# Installation

SENTRIX supports both package-based installation and direct execution.

## Option 1: Install as a CLI tool (recommended)

```bash
git clone https://github.com/parvathypjoshy/SENTRIX.git
cd SENTRIX
pip install .
sentrix example.com

## Option 2: Run directly from source

git clone https://github.com/parvathypjoshy/SENTRIX.git
cd SENTRIX
python3 sentrix.py example.com

## Requirements

Python 3.8+

Internet access for NVD / CISA KEV feeds
